BridgeTTS 使い方ガイド

日本語と英語のリアルタイム音声翻訳アプリ

← アプリに戻る

はじめに

このアプリは、日本語と英語の音声をリアルタイムで翻訳します。話した内容がテキスト化され、自動的に翻訳されます。

必要なもの: OpenAI APIキー（翻訳機能に必要）

基本的な使い方

  1. 初回起動時にOpenAI APIキーを入力して「保存して開始」をタップ
  2. 「開始」（日本語）または「Start」（英語）ボタンをタップ
  3. 「録音中」と表示されたら、マイクに向かって話すと原文と翻訳が表示されます
  4. 翻訳ボックスをタップすると、翻訳結果を音声で読み上げます（この時点で音声入力は自動的に終了します）
  5. 次の入力は、再度「開始」/「Start」ボタンをタップ（途中でやめる場合は「停止」）

ヒント: iOSでは、最初の録音開始時にマイクへのアクセス許可を求められます。「許可」をタップしてください。

ターン制について: 読み上げ（TTS）の実行は「1回の会話ターンの終了」として扱われ、音声入力は自動的に停止します。読み上げ直後にマイクを自動再開すると冒頭の音声が欠けやすいためで、日英交互の会話でも言語ボタンの押し直しで自然に切り替えられます。また、30秒間音声を検出しない場合は、電池消費と発熱を防ぐため音声入力を自動停止します。

設定項目

音声読み上げ（TTS）

翻訳結果を音声で読み上げる機能のオン/オフを切り替えられます。

停止後に自動読み上げ（会話モード）

音声入力を停止すると、最後の翻訳結果を自動で読み上げます。対面での会話でタップの手間を減らせます（初期設定はOFF）。

読み上げ速度

音声読み上げのスピードを5段階（×0.8〜×1.2）で調整できます。

文字サイズ

原文・翻訳テキストの文字サイズを小/中/大/特大から選べます。

テーマ

「自動（端末設定に追従）/ ライト / ダーク」から外観を選べます。

会話履歴

翻訳が完了すると、原文と翻訳が会話履歴として端末に保存されます（最大20件）。ヘッダーの履歴ボタン（時計アイコン）をタップすると専用画面で確認できます。

  1. 履歴の各エントリには翻訳方向（日→英 / 英→日）と時刻が表示されます
  2. 「再生」ボタンで翻訳を音声で読み上げられます
  3. 「コピー」ボタンで翻訳テキストをコピーできます
  4. 「消去」ボタンで履歴をすべて削除できます

ヒント: 履歴は端末内（localStorage）にのみ保存され、外部には送信されません。リセットボタン（オレンジ）を押しても履歴は保持されます。

iOSアプリ版（AltStore）

このアプリはPWAとしてホーム画面に追加できるほか、iOSネイティブアプリとしてもインストールできます。

  1. GitHubリポジトリのActionsから未署名IPA（BridgeTTS-unsigned.ipa）をダウンロード
  2. IPAをiPhoneに転送（AirDrop / ファイルAppなど）
  3. AltStoreの「My Apps」→「＋」からIPAを選択してインストール
  4. 初回起動時にマイクと音声認識の権限を許可

注意: 無料のApple IDで署名したアプリは7日で期限切れになります。AltStoreで定期的にRefreshしてください。詳細はリポジトリ内の docs/IOS_APP.md を参照してください。

デバウンス最適化

「デバウンス」とは、発話が区切れたと判断して翻訳を開始するまでの待ち時間です。短すぎると話の途中で翻訳が乱発され、長すぎると反応が遅く感じます。この機能は、あなたの端末・話し方での音声認識の更新間隔（話している間にテキストが更新される周期）を学習し、最適な待ち時間を言語別に算出します。

項目 説明
必要サンプル数 言語ごとに30件（日英は独立。片方の言語だけでも最適化可能）
データ収集 通常の使用中（録音中の認識結果更新時）に自動で収集
最適化アルゴリズム 更新間隔の90パーセンタイル × 1.2（外れ値に強く、通常の更新を翻訳の区切りと誤判定しない値）

ボタンの説明

ボタン 機能
最適化を実行 収集データから最適値を計算・適用
（サンプル数が足りると緑色に点滅）
設定をリセット デバウンス値をデフォルトに戻す
（収集データは保持）
完全リセット 設定と収集データの両方を削除
（初期状態に戻す）

注意: 最適化が気に入らない場合は「設定をリセット」でデフォルト値に戻せます。収集データは保持されるので、再度最適化を試すことができます。

トラブルシューティング

音声認識が動作しない

ブラウザのマイク権限を確認してください。Safari/Chromeの設定からマイクへのアクセスを許可してください。

翻訳されない

APIキーが正しく入力されているか確認してください。設定画面で再入力できます。

音声読み上げが動作しない（iOS）

iOSでは、最初の「開始」ボタンタップ時にTTSが初期化されます。録音を開始してから翻訳ボックスをタップしてください。

エラーが発生した

画面右下に「⚠️ エラーを報告」ボタンが表示されます。iOSアプリ版では内容確認のうえGitHubのIssueとして直接送信され（アカウント不要）、ブラウザ版ではIssueの下書き画面が開きます。APIキー等の機密情報は自動でマスクされます。
